Input type Format

Color: (Hexadecimal code such as #ADD8E6)

Date: (yyyy-mm-dd)

Datetime: (yyyy-mm-ddThh:mm+ff:gg, such as 2012-01-27T03:15)

Datetime-local: (yyyy-mm-ddThh:mm, such as 2012-01-27T03:15)

Month: (yyyy-mm)

Number: (Enter a number between 0 and 7)

Email: (name@domain.com)

URL: (http://www.domainname.com)

Time: (hh:mm:ss.ff)

Week: (yyyy-Wnn, such as 2012-W01)
